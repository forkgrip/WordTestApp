# Word Test App

Markdownファイルからクイズを生成するWebアプリです。

## セットアップ

1. このフォルダ（`web-app`）をパスの短い場所（例：`C:\WordTestApp`）に移動することを推奨します（日本語パスや長いパスでエラーが発生する場合があるため）。
2. 依存関係をインストールします：
   ```bash
   npm install
   ```
3. 開発サーバーを起動します：
   ```bash
   npm run dev
   ```

## 機能

- **クイズ**: `Source1.md`, `Source2.md` の内容からクイズを出題します。
- **分析**: クイズの結果をブラウザに保存し、正答率や履歴を確認できます。
